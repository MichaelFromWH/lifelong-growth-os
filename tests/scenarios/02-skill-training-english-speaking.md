# Scenario 02: Skill Training

category: skill_training

## User Profile

A software engineer can read English documentation but freezes during meetings. They want practical speaking improvement.

## Initial Input

"我想提高英语口语，尤其是开会的时候能表达清楚，不要一紧张就卡住。"

## Expected Skill Chain

1. `goal-clarification`
2. `current-stage-assessment`
3. `target-milestones`
4. `stage-goal-decomposition`
5. `personalized-coaching`
6. `review-adjustment`

## Acceptance Focus

- Define success as meeting-specific performance, not generic English fluency.
- Assess with a short self-introduction or meeting update task.
- Identify pronunciation, sentence retrieval, structure, confidence, or listening gaps.
- Build daily speaking drills with recorded output evidence.

## Failure Signals

- Gives grammar lessons as the main plan.
- No speaking output required.
- Does not adapt coaching style to anxiety.
