# Skill: Resource Curation

Use this skill when the user needs materials, examples, exercises, datasets, references, tools, people, or practice sources for the current stage.

## Purpose

Curate resources that support the current stage, not a generic list of popular materials.

## When To Use

- A confirmed current-stage task requires external material.
- The user asks what to read, watch, practice, use, inspect, or ask.
- The current material is too easy, too hard, outdated, boring, or misaligned.
- A plan needs exercises, authentic examples, tools, datasets, experts, or benchmark references.

## Process

1. Identify the exact stage action the resource supports.
2. Choose resource types: primer, reference, exercise, project example, dataset, tool, community, mentor feedback, or assessment.
3. Prefer a small number of high-signal resources.
4. Explain how to use each resource.
5. Add a stop rule so the user does not over-consume.
6. If current web lookup is required, use available browsing or search tools and cite sources.

## Output Format

```markdown
## Resource Shortlist

| Resource | Type | Supports | How To Use | Stop Rule |
| --- | --- | --- | --- | --- |

## Why These Resources

- 
```

## Quality Bar

A good resource recommendation reduces friction and increases progress. It should not become a substitute for doing the work.
